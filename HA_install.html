Messaggio dall'ideatore di Home Assistant

Installare i pacchetti:

Non installare il pacchetto network-manager.

sudo apt install -y avahi-daemon jq apparmor-utils

Eseguire come utente root:

Digitare: sudo su

Se si desidera installare Home Assistant su un PC x86 - amd64:

curl -sL https://raw.githubusercontent.com/home-assistant/supervised-installer/master/installer.sh | bash -s -- -m intel-nuc

Se si desidera installare Home Assistant su un Raspberry PI 3 ( Ubuntu 64Bit ):

curl -sL https://raw.githubusercontent.com/home-assistant/supervised-installer/master/installer.sh | bash -s -- -m raspberrypi3-64

La procedura di installazione funziona anche in modo automatico:

curl -sL https://raw.githubusercontent.com/home-assistant/supervised-installer/master/installer.sh | bash -s

Premere CTRL+D per uscire dalla shell root.

Vedere comunque la guida originale per l'installazione su diverse architetture di sistema.

Home Assistant è installato e funzionante.

Per visualizzare la pagina di registrazione utilizzare l'indirizzo "http://192.168.2.1:8123"

Se avete salvato uno snapshot di una installazione precedente di Home Assistant installare Samba, copiare il file snapshot
nella cartella backup di Home Assistant, fare click sul riquadro dello snapshot e selezionare "WIPE AND RESTORE".

In alternativa, nel caso in cui sia stato salvato uno snapshot della configurazione di Home Assistant, (Hass.io Google Drive Backup è
un'ottimo Add-On) in una cartella Google Driver, è possibile scaricare lo snapshot da Google Drive, installare Samba, copiare
il file snapshot nella cartella backup di Home Assistant, fare click sul riquadro dello snapshot e selezionare "WIPE AND RESTORE".
Al termine della procedura di ripristino, Home Assistant tornerà a ad essere uguale al momento in cui è stato salvato lo snapshot

NB: Se non è visibile il riquadro dello sanpshot da utilizzare per il ripristino, fare click sull'icona "refresh" nell'angolo
in alto a destra.
