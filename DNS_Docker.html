Docker DNS - Domain Name System

Docker isola i propri container, o programmi, dall'hardware di rete reale.
Crea una rete virtuale ed i container che necessitano di utilizzare la rete sono
obbligati ad utilizzare la rete fornita da Docker.
Solo Docker può utilizzare la rete dell'HUB.
Cosi accade anche per le macchine virtuali in VirtualBox ad esempio.

Se Docker non fosse in grado di risolvere i nomi di dominio

nameserver 8.8.8.8 nameserver 8.8.4.4 options edns0

Confermate di voler salvare il file INVIO:
Chiudete l'editor nano CTRL+X:
