arp -a

dhcp-lease-list

Utilità sistema linux


Monitorare l'HUB con Darkstat:
Statistiche con http server:

sudo apt-get install darkstat

Modificare il file di configurazione /etc/darkstat/init.cfg:

sudo nano /etc/darkstat/init.cfg

# Turn this to yes when you have configured the options below.
START_DARKSTAT=yes
# You must set this option, else darkstat may not listen to
# the interface you want
INTERFACE="-i eth1"

PORT="-p 8888"
#BINDIP="-b 127.0.0.1"
#LOCAL="-l 10.1.1.0/24"
#FIP="-f 127.0.0.1"
#DNS="-n"
#SPY="--spy eth1"

Aprire il browser e inserire l'url:

http://IP_HUB:8888
